# 📊 Sales Analytics Dashboard | SQL · Tableau · Excel

An end-to-end business analytics project that turns raw retail sales data into
clear, decision-ready insights. Data was cleaned in **Excel**, analyzed with
**SQL**, and visualized in an interactive **Tableau** dashboard.

> **Goal:** Identify what drives sales and profit across regions, product
> categories, customer segments, and discount levels — and recommend actions to
> improve profitability.

---

## 🔎 Project Overview

| | |
|---|---|
| **Domain** | Retail / E-commerce Sales |
| **Dataset** | Sample – Superstore (9,994 order rows) |
| **Tools** | SQL (SQLite), Microsoft Excel, Tableau Public |
| **Skills** | Data cleaning, exploratory analysis, KPI reporting, data visualization, business storytelling |
| **Deliverable** | Interactive dashboard + written insights & recommendations |

## ✅ Key Results 

- 📈 **~$2.3M** in total sales and **~$286K** in total profit analyzed
- 🗺️ **West** identified as the top-performing region by sales
- 💻 **Technology** delivered the highest profit of the three categories
- ⚠️ High discounts flagged as a major driver of **negative profit** in specific sub-categories (e.g., Tables, Bookcases)
