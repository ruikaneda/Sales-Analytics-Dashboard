# Images Folder

Place your dashboard screenshots here **after** you build the Tableau dashboard.

Suggested file names:
- `dashboard_overview.png` — the full dashboard
- `profit_by_subcategory.png` — the bar chart
- `sales_map.png` — the regional map

Then link them in `README.md`, for example:

```
![Dashboard Overview](images/dashboard_overview.png)
```

> 📌 Only add real screenshots of the dashboard you actually build. Do not use
> fake or borrowed images.
